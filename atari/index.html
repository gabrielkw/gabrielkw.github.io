Introdução à programação para o Atari 2600 VCS

Por Gabriel Antunes

O Atari 2600 é um video game projetado por Jay Miner e lançado em 1977 nos Estados Unidos e em 1983 no Brasil.

Hardware

A CPU

O microprocessador usado no 2600 é um MOS 6507, uma variação do popular MOS 6502 usado em produtos como o NES, SNES, Apple II e Commodore 64.

A grande diferença do 6507 é que ele possui menos pinos de endereçamento de memória (13 em vez de 16), o que permite um endereçamento de no máximo 4kB.

A CPU do 2600 roda a 1.19 MHz, um número escolhido especificamente para facilitar a sincronização dos cálculos do processador com o canhão de elétrons das televisões CRT. A cada ciclo de processamento da CPU, a televisão desenha 3 pixels na tela.

Memória

Existem dois tipos de memória bastante importantes no Atari 2600 assim como em vários outros sistemas:

  • RAM é a Memória de Acesso Aleatório, que pode ser modificada livremente, porém seu conteúdo é perdido quando a energia é desligada.
  • ROM, ou Memória Somente-Leitura, é a memória usada para guardar o código dos jogos. Essa memória não pode ser modificada.

O Atari 2600 possui ínfimos 128 bytes de RAM. O sistema é tão primitivo, no entanto, que os desenvolvedores costumavam dizer que o console possuía 1024 bits de RAM já que, se você não vai usar 8 bits de uma vez só, você não precisa dividir as informações em bytes inteiros.

O fato do processador endereçar somente 4kB fez com que a vários jogos do console fossem lançados em cartuchos de 2kB ou 4kB. Jogos mais complexos usam um sistema de troca de bancos de memória dentro do cartucho, onde o cartucho trata de trocar o grupo de informações acessado pelos registradores quando uma posição de memória específica é acessada. Você pode imaginar que a cada vez que o processo de bank switching acontece, é como se o cartucho fosse trocado por outro completamente diferente. A aplicabilidade do bank switching é teoricamente infinita e a comunidade de desenvolvimento já criou sistemas de mapeamento que permitem até 512kB de ROM e 256kB de RAM.

Um dos sistemas de bank switching mais utilizados no 2600 era o F8, onde os registradores $1FF8 e $1FF9 eram usados para acessar dois bancos de 4kB diferentes, possibilitando 8kB de dados no cartucho. Este sistema foi usado em jogos como Asteroids, Pitfall 2, Swordquest, Centipede, Ms. Pac-Man, Battlezone, Galaga e Pole Position.

TIA

TIA, ou Television Interface Adaptor, é a grande cartada do Atari 2600. Este chip foi desenvolvido por Joy Miner com o objetivo de criar uma interface entre o microprocessador e a saída serial de cor, luminosidade e sincronização dos moduladores de vídeo.
