O que são hashes?

O objetivo de uma função hash é confirmar a autenticidade de um pacote de informação (arquivos, imagens, senhas, números, etc). Uma função hash é uma função de mão única: fácil de calcular para qualquer entrada mas difícil de inverter dada a imagem de uma entrada aleatória.

A função hash discutida aqui sempre retorna um número de 6 dígitos e divide o universo em 5004 grupos diferentes.

A utilização de funções hash em salas de aula

A vantagem da utilização de funções de mão única na confirmação de resultados escolares é a possibilidade de oferecer um método de checagem para a resposta do aluno sem mostrá-lo a solução dos exercícios.

Digamos que o aluno seja apresentado ao seguinte tema de casa:

Calcule o resultado da expressão 5+7*4

hash para confirmação: 114558

Esta expressão pode causar confusão na resposta do aluno devido à ordem de operações. Suponhamos que o aluno se confunda e chegue ao resultado incorreto 48 (que seria o resultado da expressão (5+7)*4). O aluno pode, então, com a ajuda de uma calculadora simples, calcular a hash de confirmação do resultado que ele obteve. Ele perceberá a hash calculada é 125689, demonstrando um resultado incorreto.

Mesmo sabendo que não obteve êxito em sua tentativa, o aluno não conseguirá calcular a resposta certa a partir da hash apresentada no exercício.

Após uma análise mais profunda da questão, o aluno chega a resposta correta 33. Ao calcular a hash deste número, ele chega a 114558, confirmando o seu resultado.

Senhas

Imagine uma situação onde dois exercícios de dias diferentes possuem a mesma resposta. Isso faria com que as hashes dos dois exercícios fossem iguais e o aluno poderia inferir o resultado a partir disso. Senhas são a solução para isso.

Uma senha pode ser estabelecida de antemão se ela se fizer necessária. Essa senha vai fazer com que a hash seja ainda mais imprevisível.

Veja um exemplo onde dois exercícios com o mesmo resultado possuem hashes diferentes:

1. Calcule o resultado da expressão 5²

senha da hash: 412

hash para confirmação: 333457

2. Calcule o resultado da expressão 6²-11

senha da hash: 7

hash para confirmação: 114557

Como calcular uma hash manualmente

  • Insira um número na calculadora
  • Multiplique-o pela senha, se ela existir (se não, não faça nenhuma multiplicação)
  • Pressione o botão de raiz quadrada 5 vezes
  • Copie os primeiros 6 dígitos depois da vírgula que não são 0
  • Ordene os dígitos

Exemplo para o número 5638

  • Pressione o botão de raiz quadrada 5 vezes para obter o número 1,309853299712274
  • Os primeiros dígitos depois da vírgula que não são 0 são 398532
  • Ordenando os dígitos, obtemos 233589. Esta é a hash do número 5638

Exemplo para o número 654

  • Pressione o botão de raiz quadrada 5 vezes para obter o número 1,224578991764701
  • Os primeiros dígitos depois da vírgula que não são 0 são 224578
  • A hash já está ordenada. 224578 é a hash do número 654

Exemplo para o número 4987231549

  • Pressione o botão de raiz quadrada 5 vezes para obter o número 2,009361655455197
  • Os primeiros dígitos depois da vírgula que não são 0 são 936165
  • Ordenando os dígitos, obtemos 135669. Esta é a hash do número 4987231549

Exemplo para o número 2435 com a senha 54

  • Multiplique pela senha 54 para obter o número 131490
  • Pressione o botão de raiz quadrada 5 vezes para obter o número 1,445324610188779
  • Os primeiros dígitos depois da vírgula que não são 0 são 445324
  • Ordenando os dígitos, obtemos 234445. Esta é a hash do número 2435
